# docs/

Place your PDF documents here.

Examples:
- meadowridge-ccrs.pdf
- meadowridge-bylaws.pdf
- arc-application.pdf
- minutes-2025-annual.pdf
- newsletter-fall2025.pdf
- meadowridge-neighborhood-map.pdf
- budget-2025.pdf

The file names must match the `href` values in the Documents tab in `index.html`.
See the main README.md for instructions on adding new documents.
